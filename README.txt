HOW TO PUBLISH

1. Login to GitHub.
2. Click New Repository.
3. Name it: sudhir-portfolio
4. Upload all files.
5. Open Settings > Pages.
6. Source = Deploy from Branch.
7. Branch = main.
8. Save.

Your website will be live at:
https://YOURUSERNAME.github.io/sudhir-portfolio

Replace placeholders with your images and content.
